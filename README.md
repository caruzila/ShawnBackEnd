# ShawnBackEnd
BACK-END Test Lucas Oliveira

## To install required libs use:
```
pip install -r requirements.txt
```

## To run the flask server use:
```
python server.py
```


## Heroku Adress (there is nothing there):
https://shawnbackend.herokuapp.com/

## To run the test file use:
```
pytest test_github_requests.py
```

## Endpoints:

#### GetUsers:
api/users?since={numero}

#### GetUserDetails:
api/users/\<username>/details

#### GetUserRepos:
api/users/\<username>/repos
